Audio Effects

Effects change the character of your sound. You can place effects inside an "Instrument Definition" container for specific instruments, or in the setup block as Global (Master) effects.

Signal Chain Order Matters: Effects are processed from top to bottom. For example, placing Distortion before Delay creates echoes of distorted sounds; reversing them creates sharp, grainy artifacts.

1. Filter (MoogFilter)

Simulates the classic Moog low-pass filter to shape brightness.

  • Frequency [20 - 20000 Hz]: The cutoff frequency. Lower values make the sound "muffled".
  • Resonance (Q) [0.0 - 0.9]: Adds emphasis at the cutoff point. Higher values create a sharp, "electronic" character.

2. Delay

Creates repetitive echoes, adding space and depth.

  • Delay Time [0.0 - 5.0 s]: The interval between echoes.
  • Feedback: How many times the echo repeats.

3. Reverb

Simulates dense reflections in a large room, making the sound fuller and more immersive.

4. Auto-Filter

Uses an LFO to automatically sweep the filter frequency, creating "Wah-wah" or breathing effects.

  • Rate: The speed of the sweep.
  • Depth %: The range of frequency modulation.

5. Pitch-Mod

Dynamically modulates the frequency, a core tool for creating Lo-Fi character and organic movement.

  • Modulation Type:
    • Vibrato: Uses a regular sine wave. Sounds like a violin's vibrato, musical and rhythmic.
    • Jitter: Uses high-speed random noise. Sounds like "grit" or "texture" rather than pitch change.
  • Depth [Unit: Cents]:
    • In Vibrato, it's the radius of the pitch swing.
    • In Jitter, it represents the "width of pitch blur." Simulates tape motor instability for a vintage feel.

6. BitCrush

Reduces audio resolution for a raw, 8-bit vintage game sound (Lo-Fi).

7. Waveshaper

Provides warm, analog-style saturation and distortion using a tanh function.

8. Dynamics (Compressor & Limiter)

Controls volume fluctuations to prevent clipping and add "punch" to the sound.

Real-time Updates: Using effect blocks inside the draw loop or Serial events allows you to change parameters dynamically (e.g., controlling filter with a knob).

Melody List Writing Guide

The "Play Melody List" block uses a simplified text format. It supports single notes, chord names, and powerful Tie features.

1. Duration Codes

2. Modifiers

  • Rest R: e.g., RQ (Wait 1 beat).
  • Dotted .: +50% length. e.g., Q. (1.5 beats).
  • Triplet _T: 2/3 length. e.g., Q_T.
  • Tie +: Sum durations. e.g., H.+E (3.5 beats).

3. Sequential Playback

In the desktop version, melody blocks automatically queue. If you place multiple blocks, they will play one after another instead of overlapping.

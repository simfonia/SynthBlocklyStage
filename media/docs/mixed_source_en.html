Mixed Source

The Mixed Source block lets you combine a clear waveform with textured noise in a single instrument. This is the first step in creating sounds with airiness, mechanical character, or Lo-Fi vibes.

Parameters

  • Wave: Provides the stable base pitch.
    • Sine: Clean and pure.
    • Saw: Bright and aggressive.
  • Noise: Provides texture or "air".
    • White: Equal energy across all frequencies, good for simulated airflow.
    • Pink: More low-frequency energy, sounds warmer and more balanced.
  • Mix Level (Noise %): The percentage of noise in the final mix.
    • 5-15%: Subtle texture, makes the synth sound less "digital".
    • 50%+: Used for percussion-like effects or ambient sounds.

Advanced: Creating Lo-Fi Tones

Mixed Source provides the foundation. To achieve a true Lo-Fi effect, add these effects in sequence:

  1. Mixed Source (Wave + 10% Noise)
  2. Pitch-Mod: Use "Jitter" mode to add instability.
  3. Auto-Filter: Add dynamic breathing to the frequency spectrum.

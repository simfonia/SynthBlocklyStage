Super Stage Control

The stage block provides a global variable system that can be controlled via code or external inputs (MIDI/Serial).

Programmable Variables

VariableTypeDescription
waveScalefloatControls waveform jump height.
fgHuefloatControls foreground color (HSB 0-255).
masterGainfloatMaster output volume (dB).
adsrA/D/S/RfloatReal-time envelope shaping.

Keyboard Shortcuts

  • [ ↑ / ↓ ]: Octave Transpose.
  • [ ← / → ]: Switch current instrument.
  • [ Backspace ]: Reset transpose.

音源建立積木說明

在 SynthBlockly Stage 中，一切聲音的起點都在「音源建立」分類。你可以透過這些積木初始化音訊引擎、載入樣本或建立虛擬合成器。

核心規則： 必須先放入「啟動 Minim 音訊引擎」積木（通常放在 setup 區塊的最上方），其他的音訊功能才能運作。

1. 啟動 Minim 音訊引擎

功能：喚醒 Processing 的底層音訊系統。

這就像是幫音響接上電源。它會自動準備好 AudioOutput，讓所有的合成器與樣本能發出聲音。

2. 載入音訊樣本 (Sampler)

功能：將 .wav 或 .mp3 檔案載入記憶體。

  • 名稱：給這個聲音起個名字（如 "kick"）。
  • 檔案路徑：檔案必須放在專案資料夾下的 data/ 目錄中。

載入後，你可以使用演奏分類中的「觸發音訊播放」積木來播放它。

3. 建立自訂合成器

功能：透過諧波或加法合成定義音色。

這些積木必須放在「定義樂器」容器內。樂器在舞台上的切換順序會嚴格遵守積木在工作區從上到下的排列順序。

4. 選取目前演奏樂器

功能：切換發聲目標。

此積木現在具備動態感應功能，下拉選單會自動顯示你目前定義的所有樂器名稱，無需手動輸入。

波形與聽感小知識

  • 弦波 (Sine)：最純淨、柔和，像笛聲。
  • 三角波 (Triangle)：溫暖且帶有少量泛音，適合做為大部分合成器的基礎。
  • 方波 (Square)：中空、帶鼻音，適合 8-bit 電玩音效。
  • 鋸齒波 (Saw)：銳利、明亮，穿透力最強，適合 Bass 或強力的主奏。

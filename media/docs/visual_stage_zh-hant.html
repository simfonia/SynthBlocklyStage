建立表演舞台 (Super Stage)

「建立表演舞台」是本專案的靈魂積木。它不僅建立一個互動畫布，更提供了一套可透過程式即時操控的「全域變數系統」。

可程式化控制變數

你可以透過「即時表演」分類中的積木（或透過 Serial/MIDI 輸入）來改變以下變數，達到動態視覺與音效回饋：

變數名稱 類型 功能說明 建議範圍
waveScale float 控制波形跳動的高度幅度。 0.5 ~ 10.0
trailAlpha float 控制背景殘影（餘暉）。數值越低殘影越長。 0 ~ 255
fgHue float 控制前景（波形與文字）的色相 (Hue)。 0 ~ 255
masterGain float 總輸出音量（分貝）。 -40 ~ 15
pitchTranspose int 全域移調。增加 12 代表升一個八度。 -24 ~ 24
adsrA / D / S / R float 即時調整合成器的包絡線時間。 0.01 ~ 2.0

控制面板快捷鍵

在舞台啟動後，除了透過積木控制，也支援以下鍵盤操作：

  • [ Q - U ] / [ I - \ ]：演奏鋼琴音階。
  • [ ↑ / ↓ ]：調整八度移調。
  • [ + / - ]：微調半音移調。
  • [ ← / → ]：切換目前所選樂器。
  • [ Backspace ]：重設移調為 0。
專家提示： 嘗試使用 Arduino 的可變電阻 (Potentiometer) 透過 Serial 傳送數值來控制 fgHue，可以讓你在表演時用手旋轉出彩虹般的波形顏色！

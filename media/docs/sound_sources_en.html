Sound Source Creation

Everything starts here. You can initialize the audio engine, load samples, or create custom synthesizers.

Core Rule: You must place the "Init Minim Audio Engine" block (usually at the top of setup) before any other audio features can work.

1. Init Minim Audio Engine

Wakes up the underlying audio system. It prepares the AudioOutput for all sounds.

2. Load Audio Sample

Loads a .wav or .mp3 file from your project's data/ folder into memory.

3. Select Current Instrument

Switch the active target for your performance. This block automatically detects all defined instruments in your workspace.

旋律清單編寫指南

「演奏旋律清單」積木使用一種簡化的文字格式。它支援單一音符、已定義的和弦名稱，以及強大的連結線功能。

1. 時值代碼對照表 (Duration Codes)

代碼 名稱 拍數 (4/4)
W全音符4 拍
H二分音符2 拍
Q四分音符1 拍
E八分音符0.5 拍
S十六分音符0.25 拍

2. 進階符號

  • 休止符 R：如 RQ (休止一拍)。
  • 附點 .：時值增加 50%。如 Q. (1.5 拍)。
  • 三連音 _T：時值縮短為 2/3。如 Q_T。
  • 連結線 +：累加多個時值。如 H.+E (三拍半)。

3. 排隊播放機制

在桌面版中，旋律積木具備自動排隊功能：

  • 如果你連續放置多個「演奏旋律」積木，它們會一段接一段順序播放，而不會疊在一起。
  • 你可以利用這個特性，將長篇樂曲拆解成多個積木來管理。

4. 範例

// 混合單音與和弦 C4Q D4E E4E CM7H // 複雜節奏 (三拍半) C4H.+E G4Q

Performance: Triggering Sounds

These blocks push sound to the stage. We use a Note On/Off architecture, compatible with external MIDI controllers.

1. Synth Start (Note On)

Starts an oscillator and applies the ADSR Envelope. The sound will sustain until a "Stop" command is received.

2. Synth Stop (Note Off)

Triggers the Release phase of the ADSR. The sound fades out based on your Release settings.

3. Status Detection & Exit

  • Is any melody playing?: Check if background sequences are still running.
  • Exit Program: Closes the stage immediately. Combine with detection to auto-close after a song ends.

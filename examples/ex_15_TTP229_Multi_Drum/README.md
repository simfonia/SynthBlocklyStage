# Example 15: TTP229 Multi-Touch Drum Pad

此範例展示如何將 TTP229 觸控板轉換為專業的 **多鍵同時觸控 (Multi-touch)** 電子鼓墊。

## 1. 硬體設定 (重要)
為了達到最佳效能與多鍵支援，請確保 TTP229 模組上的跳線 (Jumpers) 已正確短接：

- **開啟 16 鍵模式 (TP2)**：由預設 8 鍵切換為 16 鍵。
- **開啟多鍵模式 (TP3+TP4)**：允許同時偵測多個按鍵按下。
- **提高靈敏度 (TP5)**：將採樣頻率提升至 64Hz，大幅減少打擊延遲。

## 2. 軟體實作
- **波特率 (Baud Rate)**：使用 **115200** 減少序列傳輸延遲。
- **資料格式**：採用 **16 位元狀態字串** (如 `1000000000000001`)。
- **邊緣偵測 (Edge Detection)**：Blockly 邏輯會比對當前與上一次的狀態字串，僅在按鍵「剛按下」(0 變 1) 時觸發鼓聲，按住不放不會重複觸發。

## 3. 接線圖
- VCC -> 5V
- GND -> GND
- SCL -> D2
- SDO -> D3
